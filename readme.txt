=== Syntax Code Highlighter - highlight.js ===
Contributors: L. Priori
Requires at least: 2.0.2
Tested up to: 2.1

Customizable code snytax highlighter.

== Description ==

Powerful, flexible, client-side, easy customizable code snytax highlighter. Suport style themes and easy 
customizable styles. This plugin integrates the highlight.js by softmagic into WordPress. Support 1C, AVR 
Assembler, Apache, Axapta, Bash, C#, C++, CSS, Diff, DOS .bat, Delphi, Django, HTML, XML, Ini, Java, 
JavaScript, Lisp, Lua, MEL (Maya Embedded Language), Nginx, Parser3, PHP, Perl, Python, Python profile, 
RenderMan (RIB, RSL), Ruby, Scala, SQL, Smalltalk, TeX, VBScript.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use `<pre><code>` or `[code]` in your posts
